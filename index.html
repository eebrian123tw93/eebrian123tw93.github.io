🔧 什麼是 Git Flow？

Git Flow 是一種 Git 的分支管理模型，用來組織開發流程。它主要有以下幾種分支類型：

分支名稱 用途
master 穩定的正式發佈版本
develop 開發中的主分支，準備下一個版本
feature/* 開發新功能的分支
release/* 發佈前準備的分支
hotfix/* 緊急修復生產環境的分支
